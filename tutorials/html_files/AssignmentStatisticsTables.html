Assignment Statistics Tables

Assignment Statistics Tables

The aim of the assignment statistic tables is to allow the calculation of the degree of assignment of a protein chain. From this information it can :

  • Help identify residues with missing assignments
  • Calculate Assignment percentages for atom types
  • Identify some potential assignment issues

Generated tables

AssignedChemicalShifts

Gives an overview of every chemical shift assigned from a chemical shift list for every possible atom/atom group in selected chain, consequently this gives you very large table with a very large number of columns.

Possible values:

  • Chemical shift value recorded for atom/atom group
  • empty cells indicate that no data is expected in this location.
  • ‘ - ‘ data is expected but not available for this for atom/atom group

AssignmentCompleteness

Provides an overview of which atom/atom groups are assigned.

Possible values:

  • Number of observed chemical shifts for this atom/atom group. Both integers and fractions can be observed
  • empty cells indicate that no data is expected in this location.

SummaryAssignment

Provides a list of atom types and their percentage assignment coverage.

Possible values:

  • Number assigned/unassigned
  • Percentage assigned
  • If no atoms of a type are observed then the percentage column for that value will be empty

PossibleProblems

This table only lists residues, for which recorded chemical shifts number fall outside excepted number. For instance, if Ile gamma proton has been recorded for HG1x but not for HG1y, this is listed in the HG column as 0.5.
NOTE: currently we do not catch errors such as inconsistent assignment, for instance if Ile HG1y and HG12 are both assigned then this will return 1.0

Possible values:

  • Number of observed chemical shifts for this atom/atom group. Both integers and fractions can be observed
  • empty cells indicate that no data is expected in this location.
