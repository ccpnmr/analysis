Installation

Installation on Mac or Linux

Download

You can download the software from our website at

https://www.ccpn.ac.uk/v3-software/downloads

and then go to the most recent version.

If you scroll down you will find both the Mac and the Linux download files. The Mac download does not include the tutorials and example data, so you will have to download this separately, if you would like have it. The Linux download already includes the tutorial and example data.

Installation using the command line

Once you have downloaded the program, unpack it in a directory of your choice (in this case it is ~/Applications). There are numerous ways to do this. One of them would be something like the following:

cd ~/Applications
mv ~/Downloads/download_file.zip .
unzip download_file.zip

for a .zip file or to extract a .tgz file use the following command in the final line

tar xvzf download_file.zip

The unpacking may take a minute or two. Once it is finished you should have a directory called something like ccpnmr3.0.0 (depending on the exact version you have downloaded).

Starting the program

The executable file is located in the ccpnmr3.0.0/bin directory, so you can start the program with something like the following:

cd ccpnmr3.0.0/bin
./assign

Depending on what shell you are using, you could set up an alias in your .cshrc file with the line

alias assign '~/Applications/ccpnmr3.0.0/bin/assign'

or in your .bashrc file with the line

alias assign='~/Applications/ccpnmr3.0.0/bin/assign'

Remember these will only become active if you source your .cshrc/.bashrc file or open new terminal window. Then you can start CcpNmr AnalysisAssign simply by typing

assign

at any point in a terminal.

You should now make sure that you Install Updates.
