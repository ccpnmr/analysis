Installing Updates

Once you have installed CcpNmr Analysis you should always check for Updates. Go to the Help menu and select Check for Updates.... If the number of updates is greater than 0, then click on Download and Install Updates.

When this is done, you should close CcpNmr Analysis and restart the program so that the Updates take effect.

It is worth checking for Updates every couple of week to make sure that the program is always up-to-date.
