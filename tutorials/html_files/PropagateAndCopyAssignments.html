Propagate and Copy Assignments

When you want to copy the assignments from one peak to another there are several different ways in which you can do this, depending on whether you want to take spectrum assignment tolerances into account or not or whether you want to swap assignments, or copy the assignments only from one peak to others.

Propagate Assignments

This feature can be found in the main menu under Assign / Propagate Assignments or using the shortcut PG and operates on all currently selected peaks. The feature works analogous to the V2 propagate assignments feature which essentially means that all assignments are swapped between peaks (i.e. if peak A is assigned to Ala 5 and peak B is assigned to Val 23, then all selected peaks including A and B will become assigned to both Ala 5 and Val 23). Propagating peak assignments will always take the spectrum Assignment Tolerances into account. These can be found and adjusted in the Dimensions tab of the Spectrum Properties. This means that an assignment is only copied if the source and destination dimension positions are within the Assignment Tolerance of the target Spectrum. Note, however, that the Assignment Tolerances are not taken into account if an assignment is being copied between spectra that belong to different Chemical Shift Lists. So if you have a titration series, then you should generally place all the spectra in separate Chemical Shift Lists and then you would still be able to copy the assignments using the Propagate Assignments feature.

Copy Assignments

This feature can be found in the main menu under Assign / Copy Assignments or using the shortcut CA and operates on all currently selected peaks. Like Propagate Assignments, the assignments will be swapped between all peaks, but this time the Assignment Tolerances will not be taken into account.

Propagate Assignments from Reference

This feature is available from the Selected Peak(s) context menu (i.e. when you right-click on a Peak) as Propagate Assignments from Reference. In this case the assignments will not be swapped between all peaks, but only propagated from the Peak from which the mouse context menu was accessed. Assignment Tolerances will, however, still be taken into account.

Copy Assignments from Reference

This feature is available from the Selected Peak(s) context menu (i.e. when you right-click on a Peak) as Copy Assignments from Reference. As for Propagate Assignments from Reference the assignments will be copied only from the Peak from which the mouse context menu was accessed. And as for Copy Assignments the Assignment Tolerances will not be taken into account.
