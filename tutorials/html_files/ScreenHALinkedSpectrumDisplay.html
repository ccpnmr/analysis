AnalysisScreen Hit Analysis Linked Spectrum Displays

The Hit Analysis module is usually linked to at least one SpectrumDisplay. This allows you to view a selected Peak Match automatically when selecting it in one of the tables. By default, the Hit Analysis module will be linked to the most recently opened SpectrumDisplay (Last Opened). If you go to the Settings panel and then the Appearance tab you can add or remove specific SpectrumDisplays or the Last Opened one.

In the Settings Appearance tab there are various additional options you can toggle on or off, including whether to show the spectra in stacked mode, auto-zoom on the peak or add a mark on the selected Peak Match. You can define your own preferred window size for the auto-zoom. The window width is given in ppm and the Y height factor (%) specifies the amount of white space added above and below the peak in the Y dimension. You can also set the zoom you wish to use in the SpectrumDisplay and click on Set from selected strip to apply these values to your Settings.
