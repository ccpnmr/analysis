Spectrum Calibration

Spectrum Calibration

There are several ways in which you can change your Spectrum calibration (or referencing). Which method you want to use will depend both on the circumstances of why you need to change the Spectrum referencing and on your own personal preferences.

Changing Referencing parameters manually

Go to Spectrum Properties (double-click on your Spectrum in the sidebar) and then the Dimensions tab. Change the Referencing (ppm) values by as much as required to shift the spectrum to where you want it to be. This may be a useful method if your spectrum referencing needs to be corrected by a particular ppm value.

Alternatively, change the Referencing (points) values. This might be a helpful way to re-reference your spectrum, if you know that there is a particular point offset due to the way in which the spectrum was recorded or processed.

Calibrate Spectra from Peaks

If you have two peaks in two Spectra which you know should be in the same position, then you can select them both and right-click on one of them and select Calibrate Spectra from Peaks …. You’ll get a pop-up in which you can select at the top which Peak/Spectrum should remain fixed (by default this is the one from which you selected the Calibrate Spectra from Peaks … command). Lower down you can select in which dimensions you want to move the other Spectrum so that the Peaks overlap. The original and new Peak positions and the difference (Delta) are provided for information.

Calibrate Spectra interactively

Right-click on a SpectrumDisplay and select Calibrate Spectra. The SpectrumDisplay will go into Calibration mode. You will see two thick blue lines (one horizontal and one vertical) on the SpectrumDisplay and a new panel below. With the mouse you can move the two lines revealing two thick dashed blue lines. Place the solid lines on a current position and the dashed lines on the new position where you want that current position to move to (you can also check the positions using the values in the panel below). Alternatively, type the Original Position, New Position or Delta into the boxes in the panel below and the lines in the Spectrum Display will update accordingly. Once you are ready to apply the calibration settings, click on Apply for each dimension you want to re-calibrate. Note that the calibration is applied to all Spectra currently shown in the SpectrumDisplay - you may want to toggle some off before applying the calibration. Finally, click Close to exit the Spectrum Calibration panel.
